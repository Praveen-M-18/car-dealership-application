# Car Dealership Reviews Portal

An end-to-end full-stack Django-React application designed to manage car dealerships, customer reviews, and analyze customer sentiment using IBM Watson / Cloud Natural Language Understanding APIs.

## Project Details
* **Project Name:** Car Dealership Reviews and Sentiment Analysis Portal
* **Repository Name:** car-dealership-application
* **Frameworks Used:** Django (Backend), React.js (Frontend), MongoDB (Dealer & Review databases), SQLite (Car Models database)
